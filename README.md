# Geopolitical Power Map

An interactive world map ranking nations by geopolitical standing, built as a single-file D3.js + TopoJSON application.

Live at: [geo.adityabadola.com](https://geo.adityabadola.com)

## What it does

- Visualises geopolitical power rankings for 60+ countries
- Color-coded tiers from superpowers to minor states
- Click any country to open a detailed panel covering military, economy, diplomacy, and influence
- India's boundary reflects official territorial claims including Kashmir and Aksai Chin
- Fully client-side, no backend, no dependencies to install

## Built with

- D3.js for map rendering and interactions
- TopoJSON for geographic data
- Vanilla JS + CSS, single HTML file
- Hosted on Hostinger
